# Koraci za lokalno pokretanje (UNIX/Linux):
### Predinstalirano:

- Node.js
- npm package manager
- [http-server package](https://www.npmjs.com/package/http-server)

### Terminal:
1. git clone 
2. cd SUPIT/
3. sudo npm install
3. npm run build
4. http-server -o ./ --silent
